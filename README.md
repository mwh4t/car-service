## Проект "Автосервис"

___

## Технические детали
### Среда и язык реализации
- Язык программирования: Python 3.12
- Фреймворк: Bottle 0.13.2
- Среда разработки: PyCharm 2024.3.5

___

### Установка
```
git clone https://github.com/mwh4t/car-service.git
cd car-service
pip3 install -r requirements.txt
```

___


### Использование
```
python3 app.py
```
